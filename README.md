# PRAKTIKUM-CAD-MODUL-KE-2
Praktikum CAD (Computer-Aided Design) yang berfokus pada pembelajaran dasar perancangan teknik menggunakan software SolidWorks. Dalam praktikum ini, mempelajari pembuatan sketch, pemberian dimensi dan constraint, hingga pemodelan 3D secara parametrik sesuai standar gambar teknik.
